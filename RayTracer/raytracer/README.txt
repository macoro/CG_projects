
Como compilar:

Para que funcione el codigo proporcionado debes instalar en tu sistema la libre\
ria libpng y copiar en el directorio ./src los archivos necesarios de lua (enca\
bezados y libreria) o incluir en el Makefile el lugar donde se encuentran estos\
 archivos. Puedes verificar si todo funciona ejecutando desde el directorio ../\
data el ejemplo nonhier.lua. Esto debe generar una imagen nonhier.png.
